## Element-UI Manage System

使用 vue-cli@4.x + Element-UI

## Usage

#### 涉及到的运行依赖插件：
#### 1.Element-UI 

```bash
npm i element-ui -S  
```
#### 2.px2rem-loader lib-flexble

```bash
npm install px2rem-loader lib-flexible --save
```

#### 3.Axios
```bash
npm install axios --save
```
#### 4.JS-Cookie
```bash
npm install js-cookie --save
```

#### 5.echarts

```bash
npm install echarts --save
```
